# ZA_Yukari_Infinity

『Pokémon Legends Z-A』における、ユカリトーナメントの自動周回を目的とした自動化スクリプトです。

## 概要

このスクリプトは、ホテル シュールリッシュへの移動から、施設内のユカリの元への移動、そしてトーナメント中の会話送り・バトル（技選択）を自動で繰り返します。

## 主な機能

- **自動移動機能**: 
  - マップから「ホテル シュールリッシュ」を選択して空を飛ぶ。
  - エレベーターを経由してユカリの目の前まで移動する。
- **インフィニティ・ループバトル**:
  - ZLボタンをホールドしながら、AボタンとXボタンを連打することで、会話送りとバトルを効率的に進行します。

## ファイル構成

- `ZA_Yukari_Infinity.py`: メインの周回ロジックが記述されたPythonスクリプト。

## 使い方

1. **環境の準備**:
   - PokeCon (または互換性のあるSwitch操作用Python環境) をセットアップしてください。
2. **スクリプトの配置**:
   - `ZA_Yukari_Infinity.py` を規定のコマンドディレクトリに配置します。
3. **実行方法**:
   - ゲーム内のどこからでも開始可能です（初期設定では空を飛ぶ動作から始まります）。
   - すでにユカリの前にいる場合は、コード内の初期移動メソッドの呼び出しをコメントアウトすることで、即座にバトルループを開始できます。

## 注意事項

- 本スクリプトは特定のボタン入力を規定の間隔で繰り返すものです。ゲーム内のアップデートやロード時間の変動により、動作がずれる可能性があります。
- バトルで使用する技の配置（AまたはX）を適切に設定しておいてください。
